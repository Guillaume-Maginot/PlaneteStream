Centre d’assistance vidéo

Cette page vérifie uniquement que votre navigateur peut utiliser correctement Planète Stream et ses lecteurs. Elle donne des pistes simples si une vidéo refuse de démarrer.

Important : ce diagnostic ne peut pas réparer un téléphone, modifier un navigateur ou contourner un bloqueur de contenu. Il affiche les informations utiles pour comprendre le problème et faire vos propres recherches si nécessaire.

Vérifications

Les points essentiels : connexion, sécurité, cookies, JavaScript et blocage éventuel.

Votre appareil

Ces informations aident à formuler une recherche précise. Aucune donnée n’est envoyée automatiquement.

Compatibilité vidéo

Un navigateur peut accepter certains formats et en refuser d’autres. Sur mobile, l’autoplay bloqué est normal.

Si une vidéo ne démarre pas

  1. Rechargez la page, puis relancez la vidéo.
  2. Vérifiez que JavaScript et les cookies sont autorisés.
  3. Désactivez temporairement votre bloqueur de contenu si vous en utilisez un.
  4. Essayez un autre navigateur récent, par exemple Chrome ou Firefox.
  5. Réessayez plus tard si le lecteur externe semble indisponible.

Aller plus loin

Si le problème persiste, effectuez une recherche en indiquant les éléments ci-dessous. C’est souvent plus efficace qu’une recherche vague du type “vidéo marche pas”, même si c’est tentant quand le lecteur fait le hérisson.

  • le modèle de votre téléphone ;
  • votre navigateur ;
  • le problème rencontré : écran noir, chargement infini, cookies bloqués, lecture impossible ;
  • les informations affichées dans le diagnostic.
