# Planète Stream V2.5.3 - Réponses imbriquées

## Ajout

- Le bouton Répondre est disponible sur les critiques et sur toutes les réponses.
- Les réponses peuvent donc former un vrai fil de discussion.
- Le compteur de réponses tient compte de toutes les sous-réponses.
- Les sous-réponses sont affichées récursivement avec une indentation visuelle.

## Supabase

Aucun SQL requis.
La structure existante `comments.parent_id` suffisait déjà.
